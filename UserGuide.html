Introduction

The Net Attendance Manager is a comprehensive tool designed to manage and track attendance during net sessions. It provides a user-friendly interface for recording attendance, managing sweepers, and generating reports. This guide will walk you through the features and functionalities of the application.

System Requirements

Before using the Net Attendance Manager, ensure your system meets the following requirements:

  • Operating System: Windows 10 or later
  • Python Version: Python 3.7 or later
  • Dependencies: Ensure all required Python libraries are installed. You can install them using pip install -r requirements.txt.
  • Disk Space: At least 50 MB of free disk space.
  • Screen Resolution: 1024x768 or higher.

Installation

  1. Download the Application:
    • Download the application files from the official repository.
  2. Set Up the Environment:
    • Ensure Python is installed on your system.
    • Install the required dependencies by running: pip install -r requirements.txt
  3. Run the Application:
    • Navigate to the application directory and run the following command: python NetAttendanceManager.py

Getting Started

  1. Launch the Application:
    • Double-click the NetAttendanceManager.py file or run it from the command line.
  2. Initial Setup:
    • The application will automatically set the working directory to \Net Attendance Manager on the current drive.
    • Ensure the required directories (Emergency, Past Rosters, UserGuide_files) are present in the working directory.
  3. Load Data:
    • Use the File > Open menu to load a CSV file containing attendance data.

Main Features

Menu Bar

The menu bar at the top of the application provides access to various functionalities:

  • File:
    • Open: Load a CSV file containing attendance data.
    • Save: Save the current data to a CSV file.
    • Data Management: Access the database utility for advanced data management.
    • Exit: Close the application.
  • Options:
    • Font Size: Adjust the font size of the interface (Small, Medium, Large).
    • Dark Mode: Toggle between light and dark themes.
  • Help:
    • User Guide: Open this user guide.
    • System Requirements: View the system requirements.
    • Disclaimer: Read the application disclaimer.

Top Frame

The top frame contains essential controls and information:

  • Clock: Displays the current time in 24-hour, 12-hour, or UTC format. Click to toggle between formats.
  • Start/Stop Roll Call: Begin or end the roll call session.
  • Open Net: Start a new net session.
  • Search: Search for entries in the table.
  • Stopwatch: Tracks the duration of the net session.
  • Refresh: Refresh the data in the table.

Tree View

The main table displays attendance data with the following columns:

  • Call Sign: Unique identifier for each participant.
  • QTH: Location of the participant.
  • Name: Name of the participant.
  • Status: Attendance status (Present, Absent, Excused).
  • Relayed: Indicates if the participant was relayed.
  • Traffic: Notes on traffic handled by the participant.
  • Sweep?: Indicates if the participant is a sweeper.
  • This Month: Number of times the participant has attended this month.

Actions:

  • Single-Click: Update status or other fields.
  • Double-Click: Edit cell values.
  • Right-Click: Access context menus for row manipulation.

Stats View

The stats view provides a summary of the current session:

  • Roster Count: Total number of participants.
  • Present: Number of participants present.
  • Excused: Number of participants excused.
  • Visitors: Number of visitors.
  • Traffic: Number of traffic entries.
  • Relayed: Number of relayed participants.
  • Sweep?: Number of sweepers.
  • Total Time: Duration of the net session.
  • Start Time: Start time of the session.
  • End Time: End time of the session.

Sweepers View

The sweepers view lists participants who are sweepers:

  • Call Sign: Call sign of the sweeper.
  • QTH: Location of the sweeper.
  • Name: Name of the sweeper.
  • Region: Region of the sweeper.

Actions:

  • Right-Click: Access context menus for managing sweepers.

Advanced Features

Data Management

Access advanced data management tools via the File > Data Management > Database Utility menu.

Dark Mode

Toggle between light and dark themes using the Options > Dark Mode menu.

Font Size

Adjust the font size of the interface using the Options > Font Size menu.

Troubleshooting

  • File Not Found: Ensure all required files and directories are present in the working directory.
  • Unsaved Changes: Always save your data before exiting the application.
  • Performance Issues: Close unnecessary applications to free up system resources.
