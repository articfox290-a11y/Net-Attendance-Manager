Data Base Utility System Requirements

Category Requirement Details
System Requirements Operating System
Operating System Windows 10 or later (due to `ctypes` and `subprocess` usage)
RAM 4 GB minimum, 8 GB recommended
Disk Space 50 MB (for the application and dependencies)
Display 1024x768 resolution or higher
Python Requirements Python Version
Python Version 3.7 or later
tkinter Built-in with Python, used for GUI
ttk Built-in with tkinter, used for themed widgets
simpledialog Built-in with tkinter, used for dialogs
messagebox Built-in with tkinter, used for message boxes
filedialog Built-in with tkinter, used for file dialogs
datetime Built-in with Python, used for date and time operations
csv Built-in with Python, used for CSV file handling
re Built-in with Python, used for regular expressions
ctypes Built-in with Python, used for interacting with Windows API
subprocess Built-in with Python, used for running external scripts
webbrowser Built-in with Python, used for opening web pages
Dependencies CSV Files and External Scripts
regions.csv CSV file containing city-region mappings
preamble.csv CSV file containing preamble text
PreambleRollCallNet.csv CSV file containing roll call net preamble
PreambleOpenNet.csv CSV file containing open net preamble
UserGuideDBU.html HTML file for the user guide
Zipper.py Python script for archiving files
